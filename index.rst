Linux desktop app guide
=======================

Desktop Linux can be a confusing world, with different desktops and
distributions offering their own frameworks for application authors.
This guide explains the pieces you need to make an application that works for
most desktop Linux users.

Much of this also applies to BSD and other open source platforms, but we'll
refer the 'Linux desktop' for convenience.
